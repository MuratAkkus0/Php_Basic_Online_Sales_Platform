# Food & Beverage Sales Platform (PHP)

This project is a **food and beverage sales platform** developed using **HTML, CSS, JavaScript, PHP, and SQL**.  
It does **not include any payment system integration**.

## Educational Purpose

**This project was created for educational purposes only.**  
It was built as a hands-on practice project while learning **vanilla PHP**, allowing the use of newly learned concepts in a real-worldâ€“like scenario.  
It is **not intended for production use**.

## Purpose

The main goal of this project is to demonstrate how a simple sales platform for food and beverage products can be built using **plain PHP** and a relational database, without relying on modern frameworks or external services.

## Features

### Product Management
- List food and beverage products
- Add new products
- Edit existing products
- Delete products

### Sales Platform
- Display available food and beverage items
- Basic cart-style interaction (without payment processing)
- Simple user interaction flow built with vanilla JavaScript

### Backend & Data Storage
- Server-side logic implemented with **vanilla PHP**
- Data stored and managed using **SQL**
- No external frameworks or payment gateways used

## Tech Stack

- **HTML**
- **CSS**
- **JavaScript**
- **PHP (Vanilla PHP)**
